# First steps
1. Clone the repository
2. Install dependencies
    ```
    npm install
    ```
3. NestJS is required to be installed in your machine
    ```
    npm i -g @nestjs/cli
    ```
4. Create and install database local
   ```
   docker-compose up -d
   ```
